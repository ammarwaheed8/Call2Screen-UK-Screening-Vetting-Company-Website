# Call2Screen — UK Screening & Vetting Company Website

Call2Screen is a professional website built for a UK-based recruitment screening and vetting company. The site is designed to present the company's services, build client trust, and act as a primary point of contact for businesses looking to verify and vet potential candidates.

The website reflects a clean, corporate identity suited to the compliance-driven nature of the industry — emphasising credibility, professionalism, and clarity.

**⚠️ Showcase Notice:** This is a **portfolio/showcase version** of the website. Certain interactive features (such as form submissions, javascript functionalities and backend integrations) are intentionally non-functional as the fully operational version has been delivered to the client. This repository exists purely to demonstrate the UI/UX design and front-end implementation.

## 🚀 Running Locally

No build process or dependencies required. Simply clone the repo and open the homepage:

```bash
git clone https://github.com/ammarwaheed8/Call2Screen-UK-Screening-Vetting-Company-Website.git
cd Call2Screen-UK-Screening-Vetting-Company-Website
# Open index.html in your browser
```

Or just double-click `index.html` — it works straight from the file system.
